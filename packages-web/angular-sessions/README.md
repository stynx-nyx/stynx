# @stynx-web/angular-sessions

Angular integration module for STYNX session management. Provides a standalone active-sessions component and types for account and administration screens that display or revoke user sessions.

## Usage

Import `ActiveSessionsComponent` in your Angular component or route definition:

```typescript
import { ActiveSessionsComponent } from '@stynx-web/angular-sessions';
```

See the [STYNX API reference](/docs/api-reference/stynx-web-angular-sessions) for full API details.
