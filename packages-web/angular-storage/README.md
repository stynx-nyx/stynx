# @stynx-web/angular-storage

Angular integration module for STYNX document storage. Provides upload components, service contracts, injection tokens, and XHR upload execution for applications that attach files to tenant-scoped resources.

## Usage

Import `DocumentUploadComponent` and configure the storage tokens in your Angular application:

```typescript
import { DocumentUploadComponent } from '@stynx-web/angular-storage';
```

See the [STYNX API reference](/docs/api-reference/stynx-web-angular-storage) for full API details.
