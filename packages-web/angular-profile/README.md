# @stynx-web/angular-profile

Angular integration module for STYNX user profile screens. Provides standalone profile and preference form components plus shared types for account settings and tenant-aware preference editing.

## Usage

Import `ProfileFormComponent` or `PreferencesFormComponent` in your Angular application:

```typescript
import { PreferencesFormComponent, ProfileFormComponent } from '@stynx-web/angular-profile';
```

See the [STYNX API reference](/docs/api-reference/stynx-web-angular-profile) for full API details.
