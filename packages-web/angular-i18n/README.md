# @stynx-web/angular-i18n

Angular integration module for STYNX internationalization. Provides runtime catalog loading, tenant-aware locale switching, a translate pipe, and the `StynxI18nModule` for Angular applications.

## Usage

Import `StynxI18nModule` in your Angular application module:

```typescript
import { StynxI18nModule } from '@stynx-web/angular-i18n';
```

See the [STYNX API reference](/docs/api-reference/stynx-web-angular-i18n) for full API details.
