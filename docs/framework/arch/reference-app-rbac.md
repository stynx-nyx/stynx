# Reference App RBAC Inventory

The durable machine-readable inventory lives at
`docs/framework/arch/reference-app-rbac.json`. It records the authored
roles, permission keys, endpoint bindings, Angular route bindings, and entity
bindings used to supplement DEVAI's current RBAC sensor output.

Use this page as the stable docs-site target when linking from markdown. The
JSON file remains the canonical artifact for tools and reviewers working in the
repository.
